Quick Guide: HTML and CSS

HTML Basics

  • HTML Structure: Use tags like <html>, <head>, <body> to structure your document.
  • Common Tags:
    • <h1> to <h6> for headings.
    • <p> for paragraphs.
    • <img> to display images.
    • <a> for hyperlinks.

CSS Basics

  • Selectors: Target elements using tags, classes (e.g., .classname), or IDs (e.g., #idname).
  • Properties: Use properties like color, font-size, margin, padding, and background-color.
  • Example: 
    h1 { color: blue; font-size: 2em; }

Data Visualization Tips

  • Use tools like Flourish and Google Sheets/Excel to create the visualizations
  • Keep your charts simple and avoid clutter.
  • Label axes and provide clear legends.
